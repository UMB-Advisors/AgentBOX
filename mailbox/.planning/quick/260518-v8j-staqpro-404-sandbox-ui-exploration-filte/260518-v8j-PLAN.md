<urgency_signal_contract>
Four signals, ALL derivable from row fields + one explicit fixture flag:

| Signal      | Derivation                                                                 |
|-------------|----------------------------------------------------------------------------|
| `escalate`  | `classification_category === 'escalate'`                                   |
| `aged`      | `status === 'pending'` AND now - received_at > 4 hours                     |
| `vip`       | explicit fixture flag `is_vip: true` on DraftRow (no derivation possible)  |
| `low_conf`  | `classification_confidence != null && classification_confidence < 0.75`    |

`urgencyScore(row)`: sum the signals with weights — escalate 3, vip 3, aged 2, low_conf 1.
Tiebreaker: older received_at wins. Pure function; no time-of-day weirdness.
"Urgent untouched" for the red flag = `status === 'pending' && urgencyScore > 0`.

Aggregate badge icon shows when 2+ signals fire on the same row (lucide `AlertOctagon`
or `Flame`; pick at component-write time).
</urgency_signal_contract>
