## Standing agenda (30 min)

### 1. Numbers (5 min) — GTM dashboard walkthrough
Open the dashboard. Read, don't discuss yet:
- External orders (vs target 8 by W8) · deposits collected · day-60 conversion
- Discord active members (30d) · GitHub stars · blog posts shipped (vs 12/12)
- Support sub MRR · cloud passthrough monthly total
- Kill-criteria status (any threshold near breach?)

### 2. Customer pipeline (5 min)
Who's in flight, where they're stuck.
- New deposits / orders this week
- Anyone stalled (deposit but no onboarding, onboarding but no go-live)
- Day-60 decisions coming up (continue vs opt-out, DR-40)

### 3. Content cadence (5 min)
- What shipped this week (blog / LinkedIn / X / YouTube) vs the §6.3 cadence
- What's queued for next week
- Any crisis-of-the-week piece in the 48h window (§6.5)

### 4. One decision (10 min)
The single call we need to make this week. Examples: which channel to double down on,
whether to publish a customer story, a pricing/offer tweak, a roadmap reprioritization.
- **Decision:** `____`
- **Owner:** `____`

### 5. Action items (5 min)
Owners + deadlines. Carry forward anything open from last week.
